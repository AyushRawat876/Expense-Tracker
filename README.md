# Expense Tracker

A Python Expense Tracker Application built using:

- Python
- CSV Database
- Tkinter GUI
- Matplotlib

## Features

- Add Expense
- View Expense
- Search Expense
- Update Expense
- Delete Expense

- Add Income
- View Income

- Monthly Report
- Budget Tracking

- Pie Chart Analytics

## Run

```bash
python gui.py
```
