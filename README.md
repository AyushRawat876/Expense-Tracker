# Expense Tracker

A Python-based Expense Tracker application built using Tkinter, CSV and Matplotlib.

## Features

- Add Expense
- View Expenses
- Update Expenses
- Delete Expenses
- Add Income
- View Income
- Monthly Reports
- Budget Tracking
- Pie Chart Analysis
- Bar Chart Analysis
- Tkinter GUI

## Technologies Used

- Python
- Tkinter
- CSV
- Matplotlib

## Project Structure

ExpenseTracker/
├── gui.py
├── main.py
├── database.py
├── expense_manager.py
├── income_manager.py
├── report_manager.py
├── budget_manager.py

## Run Project

```bash
python gui.py
```
